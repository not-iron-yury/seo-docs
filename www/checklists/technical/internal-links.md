# Чек-лист: Перелинковка (техническая часть)

<div class="text-block">
  <p>Здоровая техническая перелинковка сайта:</p>

- нет битых внутренних ссылок
- нет ссылок через редиректы
- нет orphan-страниц (сирот)
- все ссылки HTTPS
- все ссылки ведут на canonical-URL
</div>

<!-- --------- -->

## 1. Индексирование {#1}

<ul class='list-reset'>
  <li>1-1. Внутренние ссылки являются частью DOM, а не создаются через JS</li>
  <li>1-2. Внутренние ссылки ведут на <code>https</code> версию</li>
  <li>1-3. <code>nofollow</code> применяется только в редких технических случаях</li>
  <li>1-4. Отсутствуют внутренние ссылки со статусами <code>404</code>, <code>410</code>, <code>5xx</code></li>
  <li>1-5. Внутренние ссылки ведут на <code>canonical URL</code></li>
</ul>

<div class="text-block text-block--tip mt-30">
  <p>Внутренние ссылки должны вести на индексируемые страницы со статусом <code>200</code></p>
</div>

<!-- --------- -->

## 2. Редиректы {#2}

<ul class='list-reset'>
  <li>2-1. Внутренние ссылки не используют редирект <code>301</code>, <code>302</code>, <code>307</code></li>
  <li>2-1. Отсутствуют цепочки редиректов
  
  ```HTML
  <!-- плохо -->
  page → redirect → redirect → page
  
  <!-- допустимо -->
  page → redirect → page
  
  <!-- лучше всего -->
  page → page
  ```
  </li>
  <li>2-1. Внутренние ссылки ведут на конечный URL.</li>
</ul>

<div class="text-block text-block--tip mt-30">
  <p>Внутренние ссылки должны вести сразу на конечный URL, без редиректа.</p>
</div>

<!-- --------- -->

## 3. Консистентность с robots {#3}

<ul class='list-reset'>
  <li>3-1. Внутренние ссылки не ведут на страницы закрытые в <code>robots.txt</code></li>
  <li>3-2. Внутренние ссылки не ведут на страницы с <code>meta noindex</code>, <code>x-robots noindex</code></li>
</ul>

<div class="text-block text-block--tip mt-30">
  <p>Внутренняя перелинковка не противоречит <code>robots</code>.</p>
</div>

<!-- --------- -->

## Подводные камни {#hints}

- <strong>JavaScript-ссылки.</strong> Иногда ссылки добавляются только после JS-рендеринга страницы. Они могут не определяться краулерами.
- <strong>Краулер может начать обход не с главной.</strong> Из-за этого часть ссылок может не попасть в анализ.
