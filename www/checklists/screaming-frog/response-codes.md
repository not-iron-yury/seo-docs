# Коды ответов {#response-codes}

<ul class="info-block">
  <li class="info-block-itm">Задача: Проверить техническую целостность сайта</li>
  <li class="info-block-itm">Важность: Высокая</li>
</ul>

## Когда проверять {#when}

- редизайн
- миграция
- изменение URL

## Что проверять {#what-to-check}

- Response Codes

<p class=mt-30><i>Фильтры:</i></p>

- Redirection (3xx)
- Client Error (4xx)
- Server Error (5xx)

## Должно быть {#must-be}

- 0 внутренних ссылок на 5xx
- 0 внутренних ссылок на 4xx
- 0 цепочек редиректов
- Все редиректы → 301

## Источник проблем {#problems}

- Есть цепочки редиректов
- Неисправные внутренние ссылки
- 302 вместо 301
- Soft 404

<div class="text-block text-block--info mt-30">
  <p>Если паттерн подтверждается → потеря ссылочного веса / краулингового бюджета</p>
</div>

## Что делать {#actions}

- Обновить внутренние ссылки
- Удалить цепочки редиректов
- Исправить ошибки
