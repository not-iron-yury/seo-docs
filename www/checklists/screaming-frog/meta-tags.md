# Мета-теги (Title / Description) {#meta-tags}

<ul class="info-block">
  <li class="info-block-itm">Задача: Определить дубликацию и качество мета-тегов</li>
  <li class="info-block-itm">Важность: Высокая</li>
</ul>

## Когда проверять {#when}

- Перед релизом сайта (раздела)
- При падении SEO-трафика (возможно массовые дубли)

## Что проверять {#what-to-check}

- Page Titles
- Meta Description

## Должно быть {#must-be}

- Title и Description есть у каждой SEO-важной страницы
- все мета-теги уникальные
- мета-теги соответствуют интенту, на который отвечает страница

## Источник проблем {#problems}

- отсутствие мета-тегов у SEO-важных страниц
- дубликация
- кривая авто-генерация

<div class="text-block text-block--info mt-30">
  <p>Если паттерн подтверждается → проблемы с CTR и релевантностью</p>
</div>

## Что делать {#to-do}

- Уникализировать мета-теги
- Пересмотреть правильность шаблонов для авто-генерации
