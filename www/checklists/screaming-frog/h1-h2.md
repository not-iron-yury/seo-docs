# Заголовки H1–H2 {#h1-h2}

<ul class="info-block">
  <li class="info-block-itm">Задача: Проверить структуру контента</li>
  <li class="info-block-itm">Важность: Высокая</li>
</ul>

## Что проверять {#what-to-check}

- H1
- H2

## Должно быть {#must-be}

- уникальные H1 у каждой страницы
- H1 у каждой страницы в одном экземпляре
- H1 соответствует Title
- H2 отражают смысловую структуру страницы

## Источник проблем {#problems}

- H1 не согласован с Title
- шаблонные H1
- несколько H1
- отсутствует H1

## Действия {#actions}

- уникализировать заголовки
- согласовать заголовки с семантическим ядром
