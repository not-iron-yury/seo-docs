# Перелинковка {#internal-links}

<ul class="info-block">
  <li class="info-block-itm">Задача: Проверить структуру контента</li>
  <li class="info-block-itm">Важность: Очень высокая</li>
</ul>

## Что проверять {#what-to-check}

- Links → Crawl Depth
- Links → Inlinks
- Anchor Text

## Должно быть {#must-be}

- у SEO-важных страниц Crawl Depth ≤ 3
- SEO-важные страницы имеют много inlinks
- ссылки на важные страницы имеют анкоры с ключами

## Источник проблем {#problems}

- важные страницы находятся глубоко:
  - depth > 3 → проблема
  - depth > 4 → критично
- слабая связность между страницами
- страницы без внутренних входящих ссылок

## Действия {#actions}

- перераспределение ссылок
- упрощение структуры
