<!-- font-display - это CSS-дескриптор внутри правила @font-face, который управляет тем, как браузер ведёт себя во время загрузки веб-шрифта.

Он определяет:

показывать ли текст сразу

использовать ли временный системный шрифт

когда заменять его на загруженный веб-шрифт

Именно поэтому его упоминают в контексте метрики Cumulative Layout Shift из набора Core Web Vitals.

Что означает фраза

Web fonts не вызывают скачки layout (используется font-display)

Смысл:

шрифт загружается так, что текст отображается сразу и не вызывает сдвигов интерфейса после загрузки.

То есть:

сначала используется fallback-шрифт

после загрузки веб-шрифта текст заменяется без заметного сдвига layout

Пример использования
@font-face {
font-family: "Inter";
src: url("/fonts/inter.woff2") format("woff2");
font-display: swap;
}

Самый популярный режим - swap.

Поведение:

текст сразу показывается системным шрифтом

когда веб-шрифт загрузился - происходит замена

Основные значения font-display
swap (самый распространённый)

Текст показывается сразу fallback-шрифтом, потом меняется.

Хорошо для SEO и производительности.

block

Текст временно скрывается, пока не загрузится шрифт.

Плохо для UX.

fallback

Короткая попытка загрузить шрифт, потом fallback.

optional

Браузер может вообще не загружать шрифт, если сеть медленная.

Почему это влияет на layout

Без font-display браузер может:

сначала не показывать текст

затем отрендерить его веб-шрифтом

или

показать fallback

потом заменить шрифт с изменением размеров букв

Из-за этого элементы страницы смещаются → растёт CLS.

Практика в production

Обычно используют:

font-display: swap;

или для максимально стабильного layout:

font-display: optional;
Важный момент для frontend

Даже с swap может появляться CLS, если:

fallback-шрифт имеет другие метрики

сильно отличается x-height

Поэтому часто дополнительно используют:

size-adjust

ascent-override

descent-override

Это уже продвинутая оптимизация шрифтов для Core Web Vitals.

Если нужно, могу показать как крупные сайты (Vercel, GitHub, Shopify) настраивают шрифты так, чтобы CLS = 0 - там используется довольно интересная техника с fallback-метриками.
